**DEALMIND AI** is a cutting-edge negotiation game where your persuasion skills meet high-stakes AI commerce. Haggle with dynamic AI sellers, master different personalities, and land the deal of a lifetime—or walk away with nothing.

---

## 🚀 Key Features

- **🧠 Personality-Driven AI Sellers**: Every seller has a unique personality (Logical, Impatient, Cautious, etc.) and a dynamic mood system that reacts to your offers.
- **📊 Real-time Metrics**: Monitor **Trust** and **Patience** meters. One wrong move could end the negotiation instantly!
- **⚡ Advanced Tactics**: Don't just offer a price—**Justify** your bid, **Rapport** with the seller, **Delay** for pressure, or even try a bold **Bluff**.
- **🛍️ Interactive Storefront**: Browse high-end products across multiple categories like Tech, Footwear, and Luxury.
- **💎 Premium UI/UX**: A sleek, dark-mode professional interface built with **Framer Motion** for smooth, responsive animations and **Tailwind CSS v4** for cutting-edge styling.

## 🛠️ Getting Started

### 1. Prerequisites
- Node.js (v18+)
- MongoDB connection string
- Google Gemini API Key

### 2. Installation

Clone the repository:
```bash
git clone https://github.com/Deepakdass1326/-DEALMIND.git
cd -DEALMIND
```

**Backend Setup:**
```bash
cd server
npm install
# Create .env file and add your credentials
# PORT=5000, MONGO_URI, GOOGLE_API_KEY, JWT_SECRET
npm start
```

**Frontend Setup:**
```bash
cd client
npm install
npm run dev
```
